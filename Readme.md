# Project Setup

## 🚀 Quick Start

To get started with this project:

1. **Clone the repository**
   ```bash
   git clone https://github.com/AndresC-repo/Seminars.git
   cd Seminars/permutation-feature-importance
    or
   cd Seminars/saliency-maps
    ```

2. **Prequisites**
    Ensure [Docker](https://docs.docker.com/engine/install/) is installed and running on your system

3. **Build the environment**
   ```bash
    make build
    ```

3. **Access the notebook**
    Go into http://localhost:8889/
